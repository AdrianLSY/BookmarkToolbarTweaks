# Bookmark Toolbar Tweaks

A fork of [Bookmark Toolbar Tweaks](https://github.com/zen-browser/theme-store/tree/main/themes/ea1a5ace-f698-4b45-ab88-6e8bd3a563f0) by @n7itro, with bug fixes and streamlined options.

## Changes from original

- Fixed dark background behind bookmarks toolbar (removed forced `var(--zen-colors-tertiary)` button background)
- Removed "Enhanced transparent effects" option
- Removed "Auto-hide: Expand by hovering" option
- Removed "Auto-hide: Expand when searching" option
- Removed "Bookmarks toolbar direction" option
- Added "Hide folder name" option

## Features

### Layout

- **Center the toolbar** - Center bookmarks in the toolbar

### Icon & Text Management

- **Hide folder icons** - Remove folder icons from bookmark folders
- **Hide folder name** - Hide text labels on folder items
- **Hide website icons** - Remove favicons from bookmark items
- **Hide bookmarks name** - Show only icons without text labels

### Visual

- **Default transparent background** - Bookmarks toolbar has a transparent background
